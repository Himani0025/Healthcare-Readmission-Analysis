## Healthcare Readmission Analysis
   # Project Overview

This project analyzes hospital patient readmission patterns to identify high-risk groups and key contributing factors. The objective is to understand trends in readmission based on patient demographics and hospital-related variables using SQL, Python, and Power BI.

Hospital readmissions are a critical performance indicator in the healthcare industry, as frequent readmissions may indicate gaps in treatment, discharge planning, or patient monitoring.

# Business Objective

- Calculate overall hospital readmission rate

- Identify high-risk age groups

- Analyze department-wise readmission trends

- Perform exploratory data analysis (EDA)

- Build an interactive dashboard for insights

# Tools & Technologies Used

- MySQL – Data cleaning and querying

- Python (Pandas, NumPy, Matplotlib, Seaborn) – Data cleaning, Data analysis and EDA

- Power BI – Interactive dashboard and visualization

- Excel – Data validation
